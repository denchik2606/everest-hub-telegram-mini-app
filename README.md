# ЭВЕРЕСТ HUB - Telegram Mini App

Полноценный личный кабинет для агентов Учебного центра «Эверест» в формате Telegram Mini App.

## Главные документы

- `EVEREST_HUB_TELEGRAM_MINI_APP_PLAN.md` - Стратегический план
- `TELEGRAM_MINI_APP_TECHNICAL_GUIDE.md` - Подробная техническая инструкция с кодом
- `PROMPT_Telegram_Mini_App_Full.md` - Мощный промпт для Cursor / Claude

## Как запустить

1. Открой в Cursor
2. Скопируй промпт из `PROMPT_Telegram_Mini_App_Full.md`
3. Сгенерируй проект
4. Запусти на Vercel

Подробности в документации.
